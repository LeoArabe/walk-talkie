# 🖥️ Backend – Walk-Talkie

Este diretório contém o servidor Express da aplicação, responsável pela comunicação, autenticação e controle das sessões de conversa.

---

## 📦 Tecnologias

- Node.js
- Express
- Prisma ORM
- PostgreSQL (via Docker)
- Socket.IO (futuro)
- dotenv

---

## ▶️ Como rodar localmente

```bash
cd backend
npm install
npm run dev
```
